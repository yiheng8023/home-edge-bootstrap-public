# Public Media Kit

These synthetic, deterministic illustrations describe Home Edge Bootstrap v0.1.0 without depicting
a real router, dashboard, provider, subscriber, account, or network. The plain-text project name is
not presented as a registered trademark.

The `1280x640` hero is suitable for the GitHub social preview. Architecture, recovery, safety, and
evidence cards are available in English and Simplified Chinese; square summaries and a six-page
Chinese sequence support accessible project introductions. Every PNG has an objective text
alternative under `alt/`, a maintainable SVG source under `sources/`, and SHA-256 metadata in
`manifest.json`.

Reuse is permitted under the repository license when the image remains associated with the project
and is not used to imply vendor endorsement, a promise of compatibility, a promise of availability, or
successful account, payment, registration, financial, or regional verification. Use direct project
and release links without tracking parameters. Verify the manifest before redistribution.
