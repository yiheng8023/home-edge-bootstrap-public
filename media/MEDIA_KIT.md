# Public Media Kit

The System Atlas media set explains Home Edge Bootstrap as a technical system rather than presenting
generic promotional cards. Its print-like palette, editorial grid, topology maps, lifecycle loops,
evidence matrix, and explicit boundary labels are designed to remain useful when viewed apart from the
README.

The `1280x640` hero is suitable for the GitHub social preview. Architecture, recovery, safety, and
evidence cards are available in English and Simplified Chinese. Square summaries provide a compact
system map, while the six-page Simplified Chinese sequence covers changing reality, system topology,
lifecycle, stable state, safe exit, and adapter extension.

Every PNG has the same information hierarchy and diagram as its maintainable SVG source. Both are
generated from one deterministic PowerShell geometry model; visible copy is read from reviewed UTF-8
JSON. Each asset also has an objective plain-text alternative under `alt/` and SHA-256 metadata in
`manifest.json`. The SVG sources are self-contained and load no remote image, font, script, or tracker.

These illustrations describe project contracts. They do not depict a real router, dashboard,
provider, subscriber, account, or network, and the plain-text project name is not presented as a
registered trademark.

Reuse is permitted under the repository license when the image remains associated with the project
and is not used as evidence of vendor endorsement, universal compatibility, universal availability,
or successful account, payment, registration, financial, or regional verification. Use direct project
and release links without tracking parameters. Verify the manifest before redistribution.
